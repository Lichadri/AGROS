Contenido Data Science

El programa permitirá completar el proceso diseño y modelamiento de un problema basado en datos, lo cual incluye: Modelamiento de Problema, Diseño de Experimentos, Estrategias de Recolección de Datos, Limpieza de Datos (Data Wranling), Porpuesta de Soluciones, Medición de Performance de Algoritmos, Documentación de Experimentos, Preparación de Bancos de Pruebas, Modularización de Algoritmos, Generación de Microservicios y Visualización de Datos

A continuación detallamos el contenido de Data Science:

Tratamiento de Datos

Manejo de matrices y operaciones matriciales, Manejo de estructuras de datos y consultas, Limpieza de Datos, Lectura y escritura de formatos de datos, Manejo de series temporales y zonas horarios

Programación basada en Servicios

Manejo de estructuras de servicios web, Diseño de Pruebas, Uso de un Framework de DataOps

Visualización de Datos

Manejo de librerias de visualizacion de datos, Uso de Herramientas Interactivas
